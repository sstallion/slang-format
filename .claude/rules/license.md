# Licensing Rules

- A top-level `LICENSE` file contains the project license.
- Source code includes the copyright notice at the top of the file.
- [SPDX-License-Identifier][1] follows to identify the license.
- Save a memory for which license identifier is used for the project.
- Line comments are used to format the license block.
- A blank line follows the license block.

[1]: https://spdx.org/licenses/
